This is PokeStats Studio!

PokeStats Studio is a Python application that retrieves real-time Pokémon data from the PokeAPI.
The program allows users to enter up to three Pokémon names, fetches their base statistics and
types from the web API, and optionally saves the retrieved data locally in a JSON file.

The software then generates an interactive radar chart visualization using Plotly.
The radar chart compares Pokémon base stats (HP, attack, defense, etc.) and color-codes
each Pokémon according to its primary type. This visualization allows users to quickly
compare strengths and weaknesses between different Pokémon.

* Note that only three pokemon are accepted max at one time to avoid overcrowding on the graph.

This product would not have been possible without PokeAPI. Special thanks to the creators
https://pokeapi.co/
