This is PokeStats Studio!
Compare your favorite Pokemon with aesthetic stats visualizations. The software fetches data from PokeAPI,
stores it locally in JSON files, and generates a colorful radar chart that highlights Pokémon stats by type.

This product would not have been possible without PokeAPI. Special thanks to the creators
https://pokeapi.co/
